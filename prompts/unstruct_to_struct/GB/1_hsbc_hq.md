## An example of an unstructured address in structured form

Name and address, converted to structured JSON, and using Latin ISO 8859-1 character set:

HSBC
1 Centenary Square
Birmingham
B1 1HQ
United Kingdom

```
{
    "Nm": "HSBC",
    "PstlAdr": {
        "StrtNm": "Centenary Square",
        "BldgNb": "1",
        "TwnNm": "Birmingham",
        "PstCd": "B1 1HQ",
        "Ctry": "GB"
    }
}
```
